Core Fields
The essential questions for every entry.
Stimulus
The trigger. What just happened to make you feel an urge?
Example: "felt bored at desk", "manager sent a vague message", "saw a donut"
Thought
The story your mind is telling you right now to justify giving in.
Example: "I'll just check for 5 minutes", "I deserve a break", "I'll start my diet tomorrow"
Urge intensity
How strong the physical craving feels right now.
1 = a passing thought, 5 = strong distraction, 10 = feels impossible to resist
Action
What you actually did.
Acted: You gave in and did the habit.
Delayed: You waited 10 mins but eventually did it.
Resisted: You fought it off entirely.
Replaced: You drank water instead of snacking.
Awareness
When you realized you were in a loop.
Before: Caught the urge before doing the action.
During: Halfway through scrolling, you realized you were doing it.
After: 30 minutes later, you looked up and realized what happened.
Expanded Fields — Stimulus & Thought
Stimulus Type
Internal: A feeling from inside (e.g., stomach growl, sudden anxiety).
External: Something from outside (e.g., phone buzzed, smelled pizza).
Category
Physiological: Body needs (tired, hungry).
Emotional: Feelings (sad, stressed, lonely).
Environmental: Surroundings (walked into the kitchen).
Social: Other people (friends were drinking).
Temporal: Time-based (it's 3 PM, time for coffee).
Thought Type
Impulse: "Do it right now!"
Justification: "I worked hard, I earned this treat."
Avoidance: "This task is too hard, I'll do it later."
Emotional: "I feel terrible and this will make me feel better."
Expanded Fields — Urge & Body
Urge Type
Reward: Seeking pleasure (e.g., wanting sugar for the taste).
Escape: Running from discomfort (e.g., scrolling to ignore a difficult work task).
Relief: Calming down (e.g., biting nails due to anxiety).
Body State
Your physical battery levels.
Example: If Tension is 'High' and Energy is 'Low', you are highly vulnerable to seeking 'Escape' habits.
Emotion
Name the exact feeling in your own words.
Example: "frustrated with code", "feeling rejected", "overwhelmed"
Expanded Fields — Behavior & Outcome
Behavior Detail
Specifically what you did or what you wanted to do.
Example: "ate 4 cookies", "opened Twitter and scrolled for 25 minutes", "snapped at partner"
Delay / Effort
How long you paused, and how hard it was.
Example: "Delayed 5s", "Effort: High" means you tried hard to stop but it was difficult.
Immediate Outcome
How it felt in the exact moment of acting.
Example: 'Pleasure' (tasted good) or 'Relief' (stress faded temporarily).
Post Outcome
How you feel 10-20 minutes later.
Example: 'Regret' (feel guilty), 'Continued craving' (want more), 'Satisfaction' (glad I did it).
Expanded Fields — Context & Reflection
Context
Where were you? (e.g., 'Work', 'Home'). Who with? (e.g., 'Alone'). Were screens involved? (e.g., 'Phone').
Pattern
Have you repeated this exact loop today already? Is it a frequent habit?
Reflection
Your note to your future self. What small, practical tweak could disrupt this loop next time?
Example: "When I feel stuck on code, physically stand up and walk away for 2 mins instead of opening Reddit."
Google Sheets sync
Not configured — entries saved locally only
Paste your deployed Apps Script URL. Every new entry will sync automatically.
Step-by-step setup
  1. Step 1Go to sheets.google.com → create a new blank spreadsheet.
  2. Step 2Click Extensions → Apps Script in the top menu. Delete all the default code in the editor.
  3. Step 3Paste the code below (tap Copy code). Click the Save icon (Ctrl+S / Cmd+S).
  4. Step 4Click Deploy → New deployment. Set Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy. Authorize when Google asks.
  5. Step 5Copy the Web app URL (starts with https://script.google.com/macros/s/). Paste it in the field above. Tap Save URL.
  6. Step 6Tap Test connection. Wait 5 seconds. Check your Google Sheet — a test row should appear in row 2. If it did, you're done.
  7. Step 7Tap Push all entries to send everything already stored locally. Future entries sync automatically on save.
Apps Script code — paste into Extensions → Apps Script
// Behavior Loop Tracker — Google Apps Script receiver
// Paste this entire block. Save. Deploy as Web app.
// Execute as: Me  ·  Who has access: Anyone

const COLS = [
  'id','timestamp_ist','time_of_day','stimulus_label','thought',
  'urge_intensity','action_type','awareness','outcome_post','alternative',
  'stimulus_type','stimulus_category','thought_type',
  'urge_type','urge_duration','hunger','energy','tension',
  'fatigue','emotion','behavior_category','behavior_detail',
  'behavior_intensity','delay','effort','outcome_immediate',
  'location','social','device','repeat_today','frequency',
  'timestamp'
];

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const sheet = SpreadsheetApp
      .getActiveSpreadsheet().getActiveSheet();
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(COLS);
      sheet.getRange(1,1,1,COLS.length)
        .setFontWeight('bold').setBackground('#1a1a1a')
        .setFontColor('#ffffff');
    }
    sheet.appendRow(COLS.map(c => {
      const v = data[c];
      return (v === null || v === undefined) ? '' : String(v);
    }));
  } catch(err) { console.error(err.toString()); }
  return ContentService
    .createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService
    .createTextOutput('Loop Tracker endpoint is active.');
}

2. Upload the HTML file renamed to index.html

1. pages.cloudflare.com → Upload
2. Drop the HTML file as index.html
3. URL: yourproject.pages.dev
Install on phone as an app
  1. AndroidOpen the hosted URL in Chrome → tap the ⋮ menu → Add to Home screen. Opens full-screen, no browser bar, dark icon.
  2. iPhoneOpen the hosted URL in Safari (not Chrome) → tap the Share button → Add to Home Screen. Same result.
